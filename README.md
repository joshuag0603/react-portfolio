# React Portfolio

## Description

This Project is a Portfolio Website made using React and Deployed on Netlify. 

## Installation

Run the command "npm i" in the terminal at the root level to install the npm packages for the project.

## Usage

Click Through each of the links on the Navbar to access different pages!
